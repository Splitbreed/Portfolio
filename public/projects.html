  • mode_editSpaced Out, Project Four

    The fourth project, and second group effort, was a full CRUD application built on Ruby with an SQL database. Using postgresQL, Ruby on Rails, jQuery, Angular, and heavy styling with Materilize CSS, the website aims to provide a platform for users to find and favorite fonts. The landing pages of the site includes a jQuery plugin the queries the Google Font API and provides a multitude of choices. Two-way data binding allows the text-area to provide living, breathing feedback in terms of displaying the current font. The app is split into seperate repos, one for the front, and one for the back. Both are hosted on Heroku and are configured to speak to each other using CORS.
    Live Page | GitHub Repo (Front) | GitHub Repo (Back)
