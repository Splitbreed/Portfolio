Spaced Out

This app marks the second group effort and is a full CRUD application built on Ruby with an SQL database. Using postgresQL, Ruby on Rails, jQuery, Angular, and heavy styling with Materilize CSS, the website aims to provide a platform for users to find and favorite fonts. The landing pages of the site includes a jQuery plugin the queries the Google Font API and provides a multitude of choices. Two-way data binding allows the text-area to provide living, breathing feedback in terms of displaying the current font. The app is split into seperate repos, one for the front, and one for the back. Both are hosted on Heroku and are configured to speak to each other using CORS.

Here In My Garage Discord Bot

A personal pet project of mine, this voice chat bot is built using the Discord.js library, which is built to wrap around the Discord voice application API. The bot is based on a self-help guru named Tai Lopez, and a viral video poking fun at one of his ads. The HIMG bot was intended to be used for fun between me and friends on our personal voice channels, but I have made it completely open source and easy to operate for anyone that wants to use it. Most recently, the bot now has the ability to play music via Google Play and YouTube, the commands for which can be accessed using the "Tai Help" command list. I fix bugs and add on to this bot consistently, and find it extremely fun to work with. Feel free to join and test him!
